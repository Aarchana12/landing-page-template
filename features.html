CSS FEATURES

1. Styling and Design

  • Color and Backgrounds: Allows setting text, background, and border colors.


  • Font Styling: Supports font family, size, weight, style, and line height..

2. Selectors

  • Simple Selectors: CSS simple selectors are used to target HTML elements based on their name, class, ID, or other attributes.


  • Combinators: Select elements based on their relationship in the DOM (e.g., descendant div p).

3. Positioning and layout

  • Positioning: CSS positioning defines how elements are placed within a web page. It allows you to control the layout, stacking order, and alignment of elements.(e.g: Static, relative, absolute, fixed, and sticky)


  • Grid: A powerful system for creating complex layouts using rows and columns.
